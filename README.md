# 🚨 Improving Digital Forensic Cloud Security: A Secure Storage Model with Authentication and Optimal Key Generation Based Encryption

> 🚀 A secure cloud-based digital forensic framework designed to protect digital evidence using blockchain technology,
 authentication, and advanced encryption techniques.

---

# 🔗 📌 Project Overview

This project focuses on improving the security of digital forensic evidence stored in cloud environments. The proposed system
integrates blockchain technology, secure authentication mechanisms, optimal key generation, and advanced encryption techniques
to ensure confidentiality, integrity, and secure access to forensic data.The framework enables investigators to securely collect, 
store, verify, and retrieve digital evidence while preventing unauthorized access and data tampering.

---

- Secure user authentication and access control
